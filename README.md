# AwesomeCyan
Awesome WM configs

## Features
 1. Pre-configured wibar
 2. Bare minimum configs
 3. Lockscreen 

## Requirements 
 1. Dmenu
 2. Nitrogen, for setting wallpaper
 3. Alacritty
 4. xautolock, i3lock-color [required for lock functionality]

## Obviously you can change the configs and alter the requirements.
